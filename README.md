<H1>Виртуальный аккордеон</H1>
Программа позволяет играть на клавиатуре компьютера как на правой клавиатуре баяна / кнопочного аккордеона. 

Раскладка в которой *R*, *F*, *V* образуют хроматическую последовательность До, До#, Ре идеально ложится на клавиатуру QWERTY и точно повторяет правую клавиатуру баяна (московская **система «Б»**).

Добавлен синтезатор звука (Web Audio API), который по тембру немного напоминает баян за счет использования определенных звуковых волн.

Визуальный интерфейс — круглые кнопки, которые нажимаются синхронно с вашей реальной клавиатурой.

**Как запустить:**

1. Скачайте приложенный файл .html
2. Откройте файл в любом браузере.
3. Нажмите кнопку **«Включить звук»** (это требование браузеров) и можете играть!
